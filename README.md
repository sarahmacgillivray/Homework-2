# Training a Simple Feedforward Network

## Setup (Conda)
1) Install Miniconda or Anaconda from https://docs.conda.io/en/latest/miniconda.html  
2) Create the environment (named `machine-learning-homework-2`):  
   ```bash
   conda create -n machine-learning-homework-2 python=3.10 -y
   conda activate machine-learning-homework-2
   pip install -r requirements.txt
   ```

## Run
From the repo root, execute:
```bash
python back-propagation/autograd_demo.py
```
